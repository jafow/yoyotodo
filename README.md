# yoyo-todo
hello world demo with [yo-yo](https://github.com/maxogden/yo-yo)!

# start
```bash
$ git clone https://github.com/jafow/yoyotodo.git
$ cd yoyotodo/
$ npm install
$ npm run build
```

Open the `index.html` file in your favorite web
browser, like maybe [Beaker](https://beakerbrowser.com)!
